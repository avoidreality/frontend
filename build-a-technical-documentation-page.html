Introduction

Have you ever wanted to make a video game? Do you like playing with code?

You can make your own video game for free in your spare time with Python's pygame! You only need a working computer and an Internet connection!

Download Python

Download Python from python.org

Once you have Python installed type this code into verify the installation went correctly:

python -V

On a mac or some other unix flavor you may need to type this:

python3 -V

You should see the Python version printed and not an error. Example:

me@okcomputer ~ % python3 -V
Python 3.11.0

Install Pygame

You can install your simple (relatively) video-game engine for free with this command:

pip3 install pygame

You may not have pip3 on your operating-system (OS) and should try this command:

pip install pygame
Hello World Pygame

Please see this example from realpython on how to get started with pygame. The rest of the article this code came from is here: https://realpython.com/pygame-a-primer/

# Simple pygame program # Import and initialize the pygame library
import pygame
pygame.init()
# Set up the drawing window
screen = pygame.display.set_mode([500, 500])
# Run until the user asks to quit
running = True
while running:
# Did the user click the window close button?
for event in pygame.event.get():
if event.type == pygame.QUIT:
running = False
# Fill the background with white
screen.fill((255, 255, 255))
# Draw a solid blue circle in the center
pygame.draw.circle(screen, (0, 0, 255), (250, 250), 75)
# Flip the display
pygame.display.flip() # Done! Time to quit.
pygame.quit()
